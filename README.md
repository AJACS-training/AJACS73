# AJACS名古屋２

### 公式ウェブページ
[統合データベース講習会:AJACS名古屋２](https://events.biosciencedbc.jp/training/ajacs73)  

### 開催概要
統合データベース講習会：AJACSは、生命科学系のデータベースやツールの使い方、データベースを統合する活動を紹介する初心者向けの講習会です。  
 
### 対象
生命科学分野のデータベースを利用したい、研究に役立てたい方（初心者向け）。  

### 日時
2018年12月5日（水）　10:00-15:40
